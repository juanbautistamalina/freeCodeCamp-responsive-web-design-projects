Introducción

HTML, que significa "HyperText Markup Language" (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar utilizado para crear y diseñar páginas web. Es la piedra angular de la web y proporciona la estructura básica para el contenido en línea. HTML utiliza una serie de etiquetas y elementos para definir la estructura y el significado del contenido dentro de una página web.

¿Qué es HTML?
HTML es un lenguaje de marcado que utiliza una sintaxis simple y clara para describir la estructura de una página web. Cada elemento dentro de un documento HTML está envuelto en etiquetas, que indican al navegador cómo interpretar y mostrar ese elemento. Estas etiquetas pueden incluir contenido como texto, imágenes, enlaces, formularios y mucho más.

Importancia de HTML
HTML es fundamental para la creación de páginas web accesibles y bien estructuradas. Proporciona la base sobre la cual se construyen el diseño, la funcionalidad y la accesibilidad de un sitio web. Con HTML, los desarrolladores pueden crear páginas web que sean fáciles de leer, navegar y entender tanto para los usuarios como para los motores de búsqueda.

En resumen, HTML es el lenguaje esencial para la construcción de la web moderna. Al comprender sus fundamentos y utilizarlo correctamente, los desarrolladores pueden crear experiencias web atractivas y funcionales para una amplia audiencia en línea.

Estructura Básica

La estructura básica de un documento HTML consta de elementos fundamentales que proporcionan el esqueleto sobre el cual se construye una página web. En su forma más simple, un documento HTML comienza con la declaración <!DOCTYPE html>, seguida de la etiqueta <html> que encapsula todo el contenido de la página. Dentro de <html>, encontramos dos secciones principales: <head> y <body>.

En la sección <head>, se incluyen metadatos y referencias a recursos externos que no son visibles para los usuarios, como el título de la página, enlaces a hojas de estilo CSS, scripts JavaScript y otros elementos importantes para el funcionamiento y la presentación de la página web.

Por otro lado, la sección <body> contiene el contenido visible de la página, incluyendo texto, imágenes, enlaces, formularios y otros elementos que los usuarios pueden interactuar. Es aquí donde los desarrolladores estructuran y organizan el contenido de la página para proporcionar una experiencia de navegación intuitiva y atractiva.

Ejemplo básico de la estructura de un documento HTML:

<!DOCTYPE html>
<html>
<head>
<title>¡Mi primera página web!</title>
</head>
<body>
<h1>Bienvenido a mi página web</h1>
<p>Esta es una página web de ejemplo.</p>
</body>
</html>
Elementos Básicos

Los elementos básicos de HTML son los bloques de construcción fundamentales que se utilizan para crear una página web. Estos elementos permiten agregar contenido, estructura y funcionalidad a una página web de manera simple y efectiva.

Encabezados (<h1> - <h6>)
Los elementos de encabezado (<h1> - <h6>) se utilizan para definir títulos y subtitulos en una página web. Cuanto menor sea el número de encabezado, mayor será su importancia en la jerarquía de títulos. Por ejemplo, <h1> es el título principal de la página, mientras que <h6> se utiliza para subtitulos de menor importancia.

<h1>Este es un título principal</h1>
<h2>Este es un subtítulo</h2>
<h3>Otro subtítulo</h3>

Párrafos (<p>)
El elemento <p> se utiliza para representar un párrafo de texto en una página web. Este elemento es uno de los más comunes y se utiliza para organizar y estructurar el contenido textual de una página.

<p>Este es un párrafo de ejemplo.</p>
<p>Este es otro párrafo.</p>

Enlaces (<a>)
El elemento <a> se utiliza para crear enlaces a otras páginas web, archivos, ubicaciones dentro de la misma página, o direcciones de correo electrónico.

<a href="https://www.ejemplo.com">Ir a Ejemplo</a>
<a href="#seccion2">Ir a Sección 2</a>

Imágenes (<img>)
El elemento <img> se utiliza para incrustar imágenes en una página web. Es importante incluir el atributo src con la URL de la imagen que se va a mostrar.

<img src="imagen.jpg" alt="Descripción de la imagen">
Comentarios

Los comentarios en HTML son segmentos de texto que no se muestran en la página web final, pero que pueden ser útiles para documentar el código y hacerlo más legible para otros desarrolladores. Los comentarios en HTML se escriben entre <!-- y -->.

<!-- Este es un comentario en HTML -->
Listas y Tablas

Listas en HTML
Las listas en HTML permiten organizar y presentar información de manera estructurada. Hay tres tipos principales de listas en HTML: listas ordenadas (<ol>), listas no ordenadas (<ul>), y listas de definición (<dl>).

  • Listas Ordenadas (<ol>): Son listas numeradas, donde cada elemento de la lista se presenta con un número.
    • <ol>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
    </ol>
  • Listas No Ordenadas (<ul>): Son listas con viñetas, donde cada elemento de la lista se presenta con un símbolo de viñeta.
    • <ul>
    <li>Elemento A</li>
    <li>Elemento B</li>
    <li>Elemento C</li>
    </ul>
  • Listas de Definición (<dl>): Son listas que contienen términos y sus definiciones asociadas.
    • <dl>
    <dt>Término 1</dt>
    <dd>Definición 1</dd>
    <dt>Término 2</dt>
    <dd>Definición 2</dd>
    </dl>

    Tablas en HTML
    Las tablas en HTML permiten organizar datos en filas y columnas. La estructura básica de una tabla en HTML incluye las etiquetas <table> para definir la tabla, <tr> para las filas, <th> para las celdas de encabezado, y <td> para las celdas de datos.

    <table>
    <tr>
    <th>Encabezado 1</th>
    <th>Encabezado 2</th>
    </tr>
    <tr>
    <td>Dato 1</td>
    <td>Dato 2</td>
    </tr>
    </table>
Formularios

Los formularios HTML permiten a los usuarios enviar datos al servidor web. Se crean con la etiqueta <form> y pueden contener diferentes tipos de elementos de entrada, como campos de texto, casillas de verificación y botones de opción.

<form action="/submit_form" method="post">
<input type="text" placeholder="Nombre" required>
<input type="email" placeholder="Correo Electrónico" required>
<textarea placeholder="Mensaje" rows="4" required></textarea>
<input type="submit" value="Enviar">
</form>

En este ejemplo, los usuarios pueden ingresar su nombre, correo electrónico y un mensaje. Los datos se envían al servidor especificado en el atributo action cuando se hace clic en el botón "Enviar", utilizando el método especificado en el atributo method (en este caso, "post").

Los formularios son una parte esencial de muchas aplicaciones web, permitiendo la interacción y recopilación de información por parte de los usuarios.

Imágenes

Las imágenes son elementos visuales esenciales en una página web. Se insertan con la etiqueta <img> y tienen dos atributos principales:

<img src="ruta/a/la/imagen.jpg" alt="Descripción de la imagen">
  • src: Especifica la ubicación de la imagen.
  • alt: Proporciona un texto alternativo para la accesibilidad.
